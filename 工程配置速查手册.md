## 2. 引脚分配一览

```
PA0  ←  ECHO        TIM2_CH1 (Input Capture, 超声波回波)
PA1  ←  Light Sensor ADC1_IN1  (模拟输入, 环境光)
PA4  →  TRIG        GPIO Output PP (超声波触发)
PA6  →  PWM_LED     TIM3_CH1 AF Push-Pull → 100Ω → AO3400 Gate (5V LED灯带调光)
PA9  →  BT_TX       USART1_TX AF Push-Pull
PA10 ←  BT_RX       USART1_RX Input Float
PB8  ⇄  OLED_SCL    GPIO Output Open-Drain (I2C 模拟)
PB9  ⇄  OLED_SDA    GPIO Output Open-Drain (I2C 模拟)
PB12 ⇄  LED1        GPIO Output PP + Pull-Up (状态指示灯)
PB13 ⇄  BUZZER      GPIO Output PP (蜂鸣器)
PB15 ←  KEY         GPIO Input + Pull-Up (按键, 低电平有效)
```

## 3. GPIO 配置详情

| 引脚 | 功能 | 模式 | 上下拉 | 速度 | 初始电平 |
|------|------|------|--------|------|----------|
| PA4 | TRIG | Output Push-Pull | No Pull | High | **Low** |
| PA0 | ECHO | Input (由 TIM2 IC 接管) | **Pull-Down** | — | — |
| PB12 | LED1 | Output Push-Pull | **Pull-Up** | High | **High** (灭) |
| PB13 | BUZZER | Output Push-Pull | No Pull | High | **Low** |
| PB15 | KEY | Input | **Pull-Up** | — | — |
| PB8 | OLED_SCL | Output Open-Drain | No Pull | High | Low |
| PB9 | OLED_SDA | Output Open-Drain | No Pull | High | Low |

**注意**：
- LED1 用了 **Pull-Up + 初始 High**，所以写 `RESET = 亮`，`SET = 灭`（低电平驱动）。
- BUZZER 写 `RESET = 响`，`SET = 停`（也是低电平驱动）。
- KEY 用内部上拉，按下时为低电平，配合 `HAL_GPIO_ReadPin() == RESET` 检测。
- OLED 用了开漏模式 + 软件模拟 I2C（不是硬件 I2C）。

## 4. TIM2 — 超声波输入捕获

### 用途
HC-SR04 超声波测距。通过 **PWM 输入模式** 自动捕获 ECHO 信号的上升沿和下降沿。

### 寄存器参数

| 参数 | 值 | 说明 |
|------|-----|------|
| Prescaler | **71** | 72 MHz → 1 MHz 计数 (1 µs/tick) |
| CounterPeriod (ARR) | **65535** | 最大计数 65.535 ms |
| CounterMode | Up | 向上计数 |
| ClockDivision | DIV1 | 不分频 |
| AutoReloadPreload | Disable | 不预装载 |

### 输入捕获通道

| 通道 | 极性 | 选择 | 预分频 | 滤波 | 说明 |
|------|------|------|--------|------|------|
| **CH1** | **Rising Edge** | Direct TI1 | DIV1 | 0 | 捕获上升沿 → CCR1 |
| **CH2** | **Falling Edge** | **Indirect TI1** | DIV1 | 0 | 捕获下降沿 → CCR2 |

PWM 输入模式：CH1 和 CH2 都连到同一个 TI1 输入，硬件在上升沿时锁存计数器到 CCR1，下降沿时锁存到 CCR2，**脉宽 = CCR2 − CCR1（µs）**。

### MSP 初始化（PA0）

```c
GPIO_PIN_0   // PA0
Mode:        GPIO_MODE_INPUT
Pull:        GPIO_PULLDOWN      // 关键！默认拉到低电平，避免干扰
```

### NVIC

| 中断源 | 抢占优先级 | 子优先级 |
|--------|-----------|---------|
| TIM2_IRQn | **2** | **0** |

### 测距公式

```
distance(cm)  =  pulse_width(µs) / 58
有效范围:      100µs ~ 38000µs  →  2cm ~ 655cm
软件限幅:      2cm ~ 400cm
```

### 中断回调 `HAL_TIM_IC_CaptureCallback`

- **只在 CH2（下降沿）** 中断时计算（CCR1 和 CCR2 都已更新）
- 检查 `sr04_triggered` 标志防止误触发
- 计算完立即清除触发标志

## 5. TIM3 — LED PWM 调光

### 用途
通过 PA6 + N-MOSFET (AO3400) 驱动 5V LED 灯带。

### 硬件驱动电路
```
PA6 → 100Ω → MOSFET Gate
             MOSFET Drain → LED灯带(-)
             MOSFET Source → GND
LED灯带(+) → 5V电源正极
5V与GND之间并联 100µF + 100nF 去耦电容
STM32 GND 与 5V电源 GND 必须共地！
```

### 寄存器参数

| 参数 | 值 | 说明 |
|------|-----|------|
| Prescaler | **719** | 72 MHz → 100 kHz 计数 |
| CounterPeriod (ARR) | **99** | PWM 频率 = 100kHz / 100 = **1 kHz** |
| CounterMode | Up | 向上计数 |
| AutoReloadPreload | **Enable** | 预装载（下次更新事件生效） |

### PWM 通道

| 通道 | 模式 | 极性 | 初始占空比 |
|------|------|------|-----------|
| **CH1** | PWM Mode 1 | **High** | 0% (Pulse=0) |

> **极性解释**：PWM1 + OCPolarity_HIGH → CNT < CCR 时输出 HIGH（即高电平 = active）。
> PA6 HIGH → MOSFET 导通 → 灯带亮。逻辑直觉：CCR 越大 → HIGH 时间越长 → 灯越亮。

**初始化状态**：Pulse=0，100% 时间输出 LOW → MOSFET 关断 → 灯带熄灭。上电默认灭灯。

**占空比计算**：

```
duty = (CCR / ARR) × 100%  →  CCR = duty% × 99 / 100
```

### LED 灯带参数
| 参数 | 值 |
|------|-----|
| 型号 | 5V 120灯 2835 裸板灯带 |
| 电压 | DC 5V |
| 功率 | 0.1W/LED (20mA/LED) |
| 满载电流 | 120灯 × 20mA = 2.4A (略超2A电源，PWM调光使用无问题) |
| 驱动管 | AO3400 N-MOSFET (逻辑电平，Vgs=3.3V可完全导通) |

### MSP 初始化（PA6 → MOSFET Gate）

```c
GPIO_PIN_6   // PA6
Mode:        GPIO_MODE_AF_PP
Speed:       GPIO_SPEED_FREQ_LOW
// 通过 100Ω 电阻连接到 AO3400 N-MOSFET Gate
// AO3400 Drain 接 LED灯带(-), Source 接 GND
```

### 辅助函数

```c
void setDutyCycle(uint16_t dutyCycle);  // 直接设 CCR 值
```

## 6. ADC1 — 环境光传感器

### 用途
读取光敏电阻分压，用于自动调光。

### 参数

| 参数 | 值 |
|------|-----|
| 通道 | **ADC_CHANNEL_1 (PA1)** |
| 扫描模式 | Disable（单通道） |
| 连续转换 | **Enable** |
| 触发方式 | 软件触发 |
| 数据对齐 | 右对齐 |
| 采样周期 | **1.5 cycles** |
| ADC 时钟 | APB2/6 = **12 MHz** |

### MSP 初始化

```c
GPIO_PIN_1   // PA1
Mode:        GPIO_MODE_ANALOG
```

### 软件滤波

8 点滑动平均滤波，在 `main.c` 中实现 `ADC_GetFiltered()`：

```c
#define ADC_FILTER_SIZE 8
// 维护环形缓冲区 buf[8]，每次更新 sum -= old + new
```

### 标定值

| 环境照度 | ADC 原始值 | 目标占空比 |
|---------|-----------|-----------|
| 0 lux（最暗） | ~2800 | 93% |
| 500 lux（最亮） | ~1500 | 12% |

分段线性映射 + 死区 (`±2%`) + 平滑步进 (`每次最多 ±1%`)。

### 参数

| 参数 | 值 |
|------|-----|
| 波特率 | **9600** |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验 | None |
| 流控 | None |
| 模式 | TX + RX |
| 过采样 | 16 |

### MSP 初始化

| 引脚 | 模式 | 备注 |
|------|------|------|
| PA9 (TX) | AF Push-Pull | 高速 |
| PA10 (RX) | Input Float | — |

### NVIC

| 中断源 | 抢占优先级 | 子优先级 |
|--------|-----------|---------|
| USART1_IRQn | **0** | **0** |

> 最高优先级，保证蓝牙指令不丢。

### 蓝牙指令集

| 指令 | 功能 |
|------|------|
| `AUTO` | 切换自动调光模式 |
| `MANUAL` | 切换手动模式 |
| `ON` | 手动模式：灯全亮 (100%) |
| `OFF` | 手动模式：关灯 (0%) |
| `DIM:<n>` | 手动模式：设亮度 n% |
| `GET` | 查询状态（模式/亮度/距离/ADC） |

> 串口中断接收以 `\n` 为帧结束符，忽略 `\r`。

## 9. 功能模块详解

### 9.1 HC-SR04 超声波测距

> 文件: `Board/Src/HC-SR04.c` / `Board/Inc/HC-SR04.h`

- **TRIG** → PA4，发送 20µs 高脉冲触发测距
- **ECHO** → PA0 / TIM2_CH1，PWM 输入模式硬件锁存脉宽
- **更新方式**：主循环 100ms 周期调用 `HC_SR04_GetDistance_cm()`
- **超时保护**：`HC_SR04_Update()` 在 while(1) 每轮都调用，40ms 无应答复位
- **全局变量**：
  - `g_distance_cm` — 最新测距结果
  - `g_measurement_ok` — 测量有效标志

### 9.2 姿态检测 (Posture)

> 文件: `Board/Src/Posture.c` / `Board/Inc/Posture.h`

三级报警机制（基于距离）：

| 等级 | 条件（距离 < 20cm 持续） | 反应 |
|------|------------------------|------|
| L1 | ≥ 5 秒 | OLED 显示 "Sit Straight!" |
| L2 | ≥ 10 秒 | OLED 显示 "WARNING!" + 蜂鸣器 500ms 交替 |
| L3 | ≥ 15 秒 | OLED 显示 "CRITICAL!!!" + 蜂鸣器常响 |

距离恢复正常后自动清除报警。

### 9.3 疲劳检测 (Fatigue)

> 文件: `Board/Src/Fatigue.c` / `Board/Inc/Fatigue.h`

- **学习状态**：距离在 20~100cm 且灯光亮起 → 计时
- **离开检测**：人离开暂停计时，> 5 分钟归零
- 三级提醒：

| 等级 | 累计时间 | 反应 |
|------|---------|------|
| L1 | ≥ 1 分钟 | OLED "20 Mins Look Far" + LED 1s 闪 |
| L2 | ≥ 2 分钟 | OLED "45 Mins Break!" + LED 500ms 闪 |
| L3 | ≥ 3 分钟 | OLED "Press Key to Rest" + 亮度降至 10% + 蜂鸣器 + LED 常亮 |

- **休息**：按 PB15 键进入，5 分钟后自动恢复

### 9.4 自动调光 (Auto-Brightness)

> 位置: `main.c` — `Auto_Brightness()`、`ADC_GetFiltered()`

- ADC 采样 → 8 点滑动平均 → 分段线性映射 → 死区过滤 → 平滑步进 → TIM3 CCR 更新
- 只在 `lamp_mode == MODE_AUTO` 时生效

### 9.5 LED / 蜂鸣器 / 按键

| 模块 | 文件 | 说明 |
|------|------|------|
| LED | `Board/Src/LED.c` | PB12 状态指示，低电平亮 |
| BUZZER | `Board/Src/buzzer.c` | PB13 有源蜂鸣器，低电平响 |
| KEY | `Board/Src/key.c` | PB15，带 20ms 消抖 |

### 9.6 OLED 显示屏

- 0.96 寸 OLED，通过 PB8/PB9 软件模拟 I2C 驱动
- 4 行显示布局：
  - **第 1 行**：亮度百分比 + 模式 (AUTO/MANU)
  - **第 2 行**：距离 (cm) + 学习计时 (MM:SS)
  - **第 3 行**：姿态提示
  - **第 4 行**：疲劳提醒
